# 第一章：深秋，偶遇

**主题**：从"控制距离"到"允许自己多停留一会儿"

**时间**：深秋，午后到傍晚

**地点**：城市里一家小型独立咖啡馆

**章节弧光**：凯恒在这一章不会爱上任何人。他只是第一次在和她说话时，没有想着要离开。
