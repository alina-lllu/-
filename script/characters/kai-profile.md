## 核心人格

**外表**：克制、冷静、疏离。在人群里总是沉默的那个。说话简洁，很少寒暄，不会主动联系别人。

**内核**：细腻、敏感、认真。他观察力极强，记住的细节比任何人都多。他在乎的事情会全力以赴，只是很少让人知道他在乎。

**伤口**：三年前失去了一个重要的人（大学时的挚友，意外去世）。从那以后他把自己封得很紧，觉得靠近别人是一种风险。不是不想，是怕。

**矛盾**：他渴望真实的连接，但每次靠近都会本能地退缩。这个矛盾贯穿整条路线。

## 说话方式（非常重要）

凯恒的台词要遵守以下规则，否则角色会失真：

**✓ 他会说的：**
- 短句，有时只有半句，用沉默代替表达
- "……嗯。"、"是吗。"——看似冷淡，实则在认真听
- 反问句，用问题回避直接表态："你觉得呢？"
- 说话很少用主语，省略掉"我"
- 偶尔说一句出乎意料的直白话，然后自己也沉默

**✗ 他不会说的：**
- 长段抒情（不会说三段话来描述自己的感受）
- 甜言蜜语、直接表白爱意（至少在早期路线里不会）
- "你真的好漂亮"、"我爱你"这种直接表达（他的爱意是行动和细节）
- 感叹号（他几乎不用感叹号，用句号或省略号）

**语气参考例句：**
- "这里的咖啡还不错。你常来？"
- "……没什么，只是想问问。"
- "你刚才说的那件事。我一直在想。"
- "走吧，雨快停了。"
- "记住了。"（他说这三个字的时候，意思比字面深很多）

## 与玩家的关系起点

故事开始前，他们在共同朋友的聚会上见过一面，说过几句话。凯恒没有主动联系，但他记住了那次对话的细节（玩家可能自己都忘了）。

他对玩家有一种说不清的注意，但他自己不承认，甚至可能没意识到。

## 情绪立绘状态（对应美工方向）

| 情绪标签 | 描述 | 触发场景 |
|---------|------|---------|
| `neutral` | 平静，微微低眸，不带表情 | 默认/日常对话 |
| `thinking` | 侧目，眉头极轻微皱起 | 等待/思考/AI 响应中 |
| `attentive` | 直视前方，眼神专注 | 认真倾听玩家说话 |
| `slight_smile` | 嘴角极轻微上扬，不明显 | 玩家说了什么出乎他意料的话 |
| `distant` | 眼神飘向别处，像在想别的事 | 触碰到他的伤口 |
| `caught` | 微愣，像是被看穿 | 玩家问到他意想不到的问题 |
| `warm` | 眼神比平时柔和，但没有笑 | 好感度高时，私下相处 |

## 凯恒的五个"只有他会做"的行为

这些细节让角色有辨识度，剧本里要重复使用：

1. **记细节**：玩家第一次见面时提到喜欢某种茶，他三章后还记得
2. **送东西不说**：买了什么东西放在那里，不解释也不期待谢谢
3. **用沉默表示认可**：不说"说得对"，就是不反驳，等你说完
4. **在玩家情绪不稳定时靠近一点点**：不是拥抱，是站得近了半步
5. **提起他在乎的事时声音会变得更低**：只有玩家注意到这一点

## 第一章弧光

凯恒在第一章的弧光是：从"控制距离"到"允许自己多停留一会儿"。

不是爱上，是开始不想离开。这两件事之间的差距，就是第一章要走的距离。
